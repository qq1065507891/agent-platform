# 已完成功能与模块清单（基于代码与开发文档对照）

> 本清单依据当前代码实现整理，仅包含**已在代码中落地**的功能与模块。

## 1. 前端核心界面（MVP + 增强）

### 1.1 前端工程与基础框架
- 已初始化 `Vue 3 + TypeScript + Vite` 前端工程（`frontend`）。
- 完成路由与状态管理初始化（`vue-router`、`pinia`）。
- 接入 Arco Design 组件库与图标库。
- 配置 Vite 代理，将 `/api` 请求转发到 `http://127.0.0.1:8000`。
- 封装 Axios 网络层，支持 Token 注入、统一响应解析与 401/403 自动跳转登录。

### 1.2 全局布局与登录鉴权
- 完成后台标准布局（顶部导航 + 侧边栏 + 内容区）。
- 登录页面支持账号密码登录，调用 `POST /api/v1/auth/login`，成功后保存 Token 并进入主页。
- 未登录访问受保护页面自动跳转登录。
- 前端路由增加管理员路由守卫（`meta.requireAdmin`），非管理员禁止访问用户/角色管理页。

### 1.3 智能体市场与“我的智能体”
- 智能体市场列表页支持搜索与卡片展示。
- 点击卡片可直接进入智能体专属对话页（`/agent-chat/:agentId`）。
- 新增“我的智能体”页面（`/my-agents`）：支持只看本人智能体（`mine=true`）与创建入口。
- 新增智能体创建抽屉组件 `AgentFormDrawer`，可完成创建后刷新列表。

### 1.4 对话界面增强
- 通用会话页（`/chat/:id`）支持：
  - 左侧按智能体分组的历史会话列表；
  - 历史会话展开/收起与快速切换；
  - 头部智能体信息展示。
- 智能体专属对话页（`/agent-chat/:agentId`）支持自动创建会话并进入聊天。
- 消息发送支持流式输出（SSE），支持“停止生成”、中断态提示、滚动到底部。
- 可选开启前端流式指标日志（`VITE_ENABLE_STREAM_METRICS=true`）。

### 1.5 管理后台页面
- 用户管理页（`/admin/users`）：
  - 分页查询、关键词搜索；
  - 新增用户；
  - 角色与状态在线修改；
  - JSON 批量导入。
- 角色管理页（`/admin/roles`）：
  - 角色列表展示；
  - 权限标识可视化展示。

## 2. 后端基础能力

### 2.1 服务框架与统一响应
- FastAPI 应用已搭建，API 基础路由前缀为 `/api/v1`。
- 统一成功响应结构（`code/message/data`）。
- 全局异常处理（参数校验、HTTP 异常、未捕获异常）。

### 2.2 数据库与配置
- SQLAlchemy 基础连接与 Session 管理。
- `.env` 加载，支持通过 `DB_URL` 指定数据库连接。
- 基础配置模块（LLM 网关、模型参数、RAG/Chroma、技能沙箱超时等）。

### 2.3 认证与安全
- JWT 访问令牌生成与校验。
- 基础密码哈希与校验（SHA256）。
- Bearer Token 认证依赖（未认证拒绝访问）。

## 3. 用户、角色与权限（已落地）

### 3.1 用户登录与登出
- `POST /auth/login`：账号登录，返回 `access_token` 与用户信息。
- `POST /auth/logout`：登出接口（当前为成功返回）。

### 3.2 用户管理（管理员权限）
- `GET /users`：分页查询用户列表（支持 `keyword` 过滤）。
- `POST /users`：创建用户。
- `PUT /users/{id}`：更新用户。
- `POST /users/import`：批量导入用户。
- 用户角色与状态字段支持 `admin/user/manager` 与 `active/disabled`。

### 3.3 角色管理（管理员权限）
- `GET /roles`：查询角色列表。
- `POST /roles`：创建角色。
- `PUT /roles/{role_id}`：更新角色。
- 角色服务层已实现唯一性冲突处理与基础校验。

### 3.4 权限授予（管理员权限）
- `POST /permissions/grant`：授权接口（支持 subject/object/actions）。
- 授权操作会写入 `EventLog` 事件审计记录。

## 4. 智能体管理（MVP）

### 4.1 智能体 CRUD
- `GET /agents`：分页查询智能体列表（支持 `keyword`、`is_public`、`mine` 过滤）。
- `POST /agents`：创建智能体。
- `GET /agents/{id}`：获取智能体详情。
- `PUT /agents/{id}`：更新智能体（仅创建者或管理员）。

### 4.2 智能体数据模型
- Agent 模型包含：名称、描述、Prompt 模板、skills、是否公开、状态、owner_id。
- `skills` 字段支持 JSON 列表并进行基础校验。

## 5. 对话与会话（MVP + 流式）

### 5.1 会话管理
- `POST /conversations`：创建会话。
- `GET /conversations`：查询当前用户会话列表（支持按 `agent_id` 过滤）。
- `GET /conversations/{id}`：获取会话详情。
- 会话仅允许所属用户访问。

### 5.2 消息发送与多轮对话
- `POST /conversations/{id}/messages`：发送消息并返回助手回复与 trace_id。
- `POST /conversations/{id}/messages/stream`：SSE 流式回复。
- 会话消息存储在数据库 JSON 字段中。
- 支持多轮上下文拼接。

## 6. Agent 编排、工具与外部技能

### 6.1 LangGraph 编排
- 使用 `StateGraph` 构建对话流程。
- 支持 AIMessage `tool_calls` 分支与工具回路调用。
- 提供流式与回退策略（graph stream / direct LLM fallback）。

### 6.2 内置 Skills（已实现）
- `calculator`：四则运算工具。
- `current_time`：获取当前时间工具。
- `retriever_tool`：检索知识库片段并注入上下文。

### 6.3 外部 Skills 管理（已实现基础链路）
- `POST /skills/load`：创建外部技能加载任务（当前支持通过 `source_url` 下载）。
- `POST /skills/{skill_id}/disable`：禁用技能。
- `GET /skills`：技能列表（内置 + 数据库技能合并返回，分页结构）。
- 已接入 Celery 任务入口（`load_external_skill_task`）与沙箱执行模块。

## 7. 知识库 / RAG（已落地基础能力）

### 7.1 知识上传与入库
- `POST /knowledge/upload`：支持上传文件并入库向量索引。
- 支持 `agent_id` 维度绑定知识数据。

### 7.2 文档处理与检索
- 支持 `pdf/doc/docx/txt` 等文本抽取。
- 使用 `RecursiveCharacterTextSplitter` 做分块。
- 基于 Chroma 向量库存储与检索（支持本地持久化或远端连接）。

## 8. 数据模型与 Schema

### 8.1 已实现数据模型
- `User`、`Role`、`PermissionGrant`、`Agent`、`Conversation`、`Skill`。
- `RequestLog`、`LLMUsage`、`SkillInvocation`、`EventLog` 等日志/审计模型已定义。

### 8.2 Pydantic Schema
- 用户、角色、权限、智能体、会话、技能、认证、通用响应等 Schema 已齐全。
